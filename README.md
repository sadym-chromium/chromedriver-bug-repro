# ChromeDriver Bug Repro Templates

This template repository has a collection of ChromeDriver regression test templates
for different operating system and technology combinations.

## Using this repo

1.  Click the "Use this template" button and select "Create a new repository."
2.  Select an owner for the new repository, and enter a repository name.
3.  Choose the visibility of the new repository (Public or Private).
4.  Click "Create repository from template."
5.  Clone the newly created repository to your local machine.
6.  Follow the instructions in the README.md file of the specific template you want to use to reproduce a bug.
7.  Included GitHub action workflows will setup the environment and run the script to reproduce the bug, and collect ChromeDriver verbose log in the action log.
